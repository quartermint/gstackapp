## Integration Check

**3 bugs found and fixed during audit:**

1. **Manual refresh skipped health phase** — `POST /api/projects/refresh` didn't pass `sqlite` arg, silently skipping the entire health engine on user-triggered scans. Fixed.

2. **MCP sync_status wrong check type** — `"deleted_branch"` didn't match canonical `"remote_branch_gone"`, silently excluding remote-branch-gone findings from sync reports. Fixed.

3. **GitHub-only projects showed green dot** — No `host === "github"` guard in projects route, so unmonitored projects displayed as healthy instead of gray/unmonitored (HLTH-10). Fixed.

**1 dead code item (accepted):**
- `onCopyDiverged` SSE hook callback defined but never connected in App.tsx. Functionally covered by `onHealthChanged` → `refetchRisks` path. Left as a valid extension point for future use.
