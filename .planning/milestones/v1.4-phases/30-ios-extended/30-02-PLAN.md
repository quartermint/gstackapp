Add voice recording with live waveform visualization and on-device transcription that chunks at 55-second intervals. After recording, user previews transcription and saves both the audio file and transcription text to the offline queue.

```swift
import AVFoundation

@MainActor
@Observable
final class AudioRecorderService {
    private(set) var isRecording = false
    private(set) var audioURL: URL?
    private(set) var duration: TimeInterval = 0
    private(set) var currentLevel: Float = -160 // dB, for waveform

    private var audioRecorder: AVAudioRecorder?
    private var meteringTimer: Timer?
    private var durationTimer: Timer?
}
```

Methods:
- `func startRecording() throws -> URL`:
  1. Set audio session: `AVAudioSession.sharedInstance().setCategory(.playAndRecord, mode: .default, options: [.defaultToSpeaker])` then `.setActive(true)` (per research Pitfall 3)
  2. Create directory: `FileManager.default.containerURL(forSecurityApplicationGroupIdentifier: AppConstants.appGroupID)!/AudioCaptures/`
  3. File URL: `containerURL.appendingPathComponent("\(UUID().uuidString).m4a")`
  4. Settings: `AVFormatIDKey: kAudioFormatMPEG4AAC, AVSampleRateKey: 44100, AVNumberOfChannelsKey: 1, AVEncoderAudioQualityKey: AVAudioQuality.high.rawValue`
  5. `audioRecorder = try AVAudioRecorder(url:settings:)`
  6. `audioRecorder.isMeteringEnabled = true` (for waveform via averagePower, per research recommendation over AVAudioEngine)
  7. `audioRecorder.record()`
  8. Start metering timer (0.05s interval) updating `currentLevel` from `audioRecorder.averagePower(forChannel: 0)` after calling `updateMeters()`
  9. Start duration timer (1.0s interval) incrementing `duration`
  10. Set `isRecording = true`, store `audioURL`
  11. Return the file URL

- `func stopRecording()`:
  1. `audioRecorder?.stop()`
  2. Invalidate both timers
  3. Set `isRecording = false`
  4. Deactivate audio session: `try? AVAudioSession.sharedInstance().setActive(false)`

- `func deleteRecording()`:
  1. If audioURL exists, `try? FileManager.default.removeItem(at: audioURL)`
  2. Reset audioURL, duration, currentLevel

**3. Create TranscriptionService.swift** (per D-06, D-09, research Pitfall 2):

```swift
import Speech
import AVFoundation

@MainActor
@Observable
final class TranscriptionService {
    private(set) var isTranscribing = false
    private(set) var chunks: [String] = []
    private(set) var currentPartialText: String = ""

    var fullTranscription: String {
        let committed = chunks.joined(separator: " ")
        if currentPartialText.isEmpty { return committed }
        if committed.isEmpty { return currentPartialText }
        return committed + " " + currentPartialText
    }

    private let speechRecognizer: SFSpeechRecognizer?
    private let audioEngine = AVAudioEngine()
    private var recognitionRequest: SFSpeechAudioBufferRecognitionRequest?
    private var recognitionTask: SFSpeechRecognitionTask?
    private var chunkTimer: Timer?
}
```

Methods:
- `static func requestPermission() async -> Bool`:
  Request SFSpeechRecognizer authorization. Return true if `.authorized`.

- `func startTranscribing() throws`:
  1. Guard `speechRecognizer?.isAvailable == true && speechRecognizer?.supportsOnDeviceRecognition == true` else throw `TranscriptionError.notAvailable`
  2. Set `isTranscribing = true`
  3. Call `startNewChunk()`

- `private func startNewChunk()`:
  1. Create `SFSpeechAudioBufferRecognitionRequest()` with `requiresOnDeviceRecognition = true` and `shouldReportPartialResults = true`
  2. Install tap on `audioEngine.inputNode` (bus 0, bufferSize 1024, format from outputFormat)
  3. Tap callback: `request.append(buffer)`
  4. `audioEngine.prepare()` then `try audioEngine.start()`
  5. `recognitionTask = speechRecognizer.recognitionTask(with: request)` with result handler:
     - On partial result: update `currentPartialText`
     - On `isFinal`: append to `chunks`, clear `currentPartialText`
  6. Schedule `chunkTimer` at 55 seconds (not repeating) that calls `restartChunk()` (per research: 5-second safety margin before 60-second limit)

- `private func restartChunk()`:
  1. Invalidate chunkTimer
  2. Remove tap from inputNode
  3. Stop audioEngine
  4. `recognitionRequest?.endAudio()` (triggers isFinal callback)
  5. Short delay (0.1s) for final callback, then `startNewChunk()`

- `func stopTranscribing()`:
  1. Invalidate chunkTimer
  2. Remove tap, stop audioEngine
  3. `recognitionRequest?.endAudio()`
  4. Set `isTranscribing = false`

- `func reset()`: Clear chunks, currentPartialText, set isTranscribing to false

Define `enum TranscriptionError: Error { case notAvailable, permissionDenied }`

Init: `speechRecognizer = SFSpeechRecognizer()` (defaults to device locale)

```swift
final class VoiceCaptureViewModel {
```
