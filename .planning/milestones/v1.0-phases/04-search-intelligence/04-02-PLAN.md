    1. Smart detection heuristic (export needsAIRewrite):
       - QUESTION_WORDS regex: /^(what|when|where|who|how|why|which|find|show|get)\b/i
       - QUESTION_PATTERNS regex: /\?$|did i|was i|about the|related to|working on|thinking about/i
       - Logic: 1-2 words without question word -> false. Question word or pattern match -> true. 3+ words -> true.

    2. Query rewrite schema (Zod):
       ```
       queryRewriteSchema = z.object({
         ftsQuery: z.string() -- optimized FTS5 search terms
         projectFilter: z.string().nullable() -- project slug or null
         typeFilter: z.enum(["capture", "commit", "project"]).nullable()
         dateFilter: z.object({ after: z.string().nullable(), before: z.string().nullable() }).nullable()
         reasoning: z.string() -- brief explanation
       })
       ```

    3. rewriteQuery function:
       - Takes naturalLanguageQuery and projects array (slug, name, tagline)
       - Uses generateText + Output.object + google(modelId) pattern from ai-categorizer.ts
       - Prompt: "You are a search query optimizer for a personal project management system. Given a natural language question, extract: (1) key FTS5 search terms, (2) project filter, (3) type filter, (4) date range. Available projects: [list]. User query: [query]."
       - Returns QueryRewriteResult or fallback { ftsQuery: rawQuery, projectFilter: null, typeFilter: null, dateFilter: null, reasoning: "AI returned no output" }

    4. processSearchQuery orchestrator (main export):
       - Takes rawQuery and projects array
       - If !needsAIRewrite(rawQuery) || !isAIAvailable(): return { ftsQuery: sanitizeFtsQuery(rawQuery), filters: null, rewritten: false }
       - Try rewriteQuery. On success: return { ftsQuery: sanitizeFtsQuery(result.ftsQuery), filters: { project, type, dateAfter, dateBefore }, rewritten: true, reasoning }
       - On error: fallback to { ftsQuery: sanitizeFtsQuery(rawQuery), filters: null, rewritten: false }
       - Import sanitizeFtsQuery from search.ts (export it if not already exported)
