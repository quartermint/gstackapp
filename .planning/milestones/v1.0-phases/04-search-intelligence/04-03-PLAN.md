       - Interface SearchFilters: { project: string | null, type: string | null, dateAfter: string | null, dateBefore: string | null }
       - Interface SearchState: { results: SearchResult[], filters: SearchFilters | null, rewrittenQuery: string | null, loading: boolean, query: string }
       - SearchResult interface matching API response: { id, content, snippet, sourceType, sourceId, projectSlug, rank, createdAt }
       - useSearch(query: string, enabled: boolean): SearchState
         Debounced fetch (200ms) to /api/search?q={query}. Parses response into results + filters + rewrittenQuery. Returns loading state. Cancels in-flight requests on new query (AbortController). Resets to empty when query is empty or !enabled.
       - Export removeFilter function: takes current filters and key, returns new filters with that key set to null. Used by filter chip dismiss.

         b. Snippet: parse with parseSnippet, render <mark> tags as <span className="bg-yellow-200/40 dark:bg-yellow-500/30 font-medium"> for highlight.
         c. Relative timestamp: right-aligned, muted, small text. Use formatRelativeTime from lib/time.ts.
       - Wrap in Command.Item with onSelect handler and cmdk-item className.
