    **2. Create capture submission hook** (`packages/web/src/hooks/use-capture-submit.ts`):
    - Export `useCaptureSubmit(onSuccess?: () => void): { submit: (rawContent: string) => void; isPending: boolean }`
    - POST to `/api/captures` with `{ rawContent }` body
    - Set `isPending` true during request, false after
    - Call `onSuccess` callback after successful submission
    - On error: console.error (don't crash -- capture field should stay usable)
    - Do NOT use useOptimistic for v1 -- keep it simple. TanStack Query can be added later if needed.

    **3. Create captures data hook** (`packages/web/src/hooks/use-captures.ts`):
    - Export `useCaptures(projectId?: string): { captures: CaptureItem[]; total: number; loading: boolean; refetch: () => void }`
      - Fetches from `GET /api/captures?projectId=xxx&limit=5` (5 most recent per project)
      - If no projectId, fetches unlinked captures (where projectId is null -- use `GET /api/captures?status=enriched&limit=10` and filter client-side for null projectId, OR add a dedicated endpoint)
      - Pattern: follow the same fetch + useState + useEffect pattern from `use-projects.ts`
    - Export `useRecentCaptures(limit?: number): { captures: CaptureItem[]; loading: boolean }`
      - Fetches `GET /api/captures?limit={limit}` (default 3) for command palette suggestions
    - Export `useStaleCount(): { count: number; loading: boolean }`
      - Fetches `GET /api/captures/stale` (from Plan 01) and returns `.total` for the triage badge
    - Define `CaptureItem` interface locally (matching API response shape):
      ```typescript
      interface CaptureItem {
        id: string;
        rawContent: string;
        type: "text" | "voice" | "link" | "image";
        status: "raw" | "pending_enrichment" | "enriched" | "archived";
        projectId: string | null;
        aiConfidence: number | null;
        linkUrl: string | null;
        linkTitle: string | null;
        linkDescription: string | null;
        linkDomain: string | null;
        createdAt: string;
        updatedAt: string;
      }
      ```
    - Include a `refetch` function that re-triggers the fetch (set loading, re-run effect via counter state)

    **4. Create keyboard shortcuts hook** (`packages/web/src/hooks/use-keyboard-shortcuts.ts`):
    - Export `useKeyboardShortcuts(handlers: { onCmdK: () => void; onSlash: () => void; onEscape: () => void }): void`
    - Add global `keydown` listener via useEffect with cleanup
    - Cmd+K (or Ctrl+K on non-Mac): call `onCmdK`, preventDefault
    - `/` key: call `onSlash` ONLY if `document.activeElement` is NOT an input, textarea, or contenteditable element. This prevents the shortcut from firing when user is typing in the capture field or palette.
    - Escape: call `onEscape`
    - All handlers are latest-ref stable (use useRef + useEffect pattern to avoid stale closures)
