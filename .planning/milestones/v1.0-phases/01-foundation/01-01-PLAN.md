  <action>
    Create the shared contracts that both API and web packages consume. These are the type-level foundation for the entire system.

    **`packages/shared/src/schemas/capture.ts`:**
    - `captureSchema`: Full capture object shape -- `id` (string), `rawContent` (string, min 1, max 10000), `type` (enum: text, voice, link, image -- default text), `status` (enum: raw, pending_enrichment, enriched, archived -- default raw), `projectId` (string nullable), `userId` (string nullable -- PLAT-02 multi-user future-proofing), `createdAt` (string ISO datetime), `updatedAt` (string ISO datetime).
    - `createCaptureSchema`: Input for creating a capture -- `rawContent` (required), `type` (optional, default text), `projectId` (optional), `userId` (optional), `clientId` (optional string -- for idempotency).
    - `updateCaptureSchema`: Partial update -- `rawContent` (optional), `type` (optional), `status` (optional), `projectId` (optional).
    - `captureIdSchema`: `z.object({ id: z.string().min(1) })` for path params.
    - `listCapturesQuerySchema`: `z.object({ projectId: z.string().optional(), status: z.enum([...]).optional(), userId: z.string().optional(), limit: z.coerce.number().int().min(1).max(100).default(50), offset: z.coerce.number().int().min(0).default(0) })`.

    **`packages/shared/src/schemas/project.ts`:**
    - `projectSchema`: Full project shape -- `slug` (string), `name` (string), `tagline` (string nullable), `path` (string), `host` (enum: local, mac-mini), `branch` (string nullable), `dirty` (boolean nullable), `dirtyFiles` (string array), `lastCommitHash` (string nullable), `lastCommitMessage` (string nullable), `lastCommitTime` (string nullable), `lastScannedAt` (string ISO datetime nullable), `createdAt` (string), `updatedAt` (string).
    - `projectSlugSchema`: `z.object({ slug: z.string().min(1) })`.
    - `projectListQuerySchema`: `z.object({ host: z.enum(['local', 'mac-mini']).optional(), userId: z.string().optional() })`.

    **`packages/shared/src/schemas/api.ts`:**
    - `apiErrorSchema`: `z.object({ error: z.object({ code: z.string(), message: z.string(), details: z.array(z.unknown()).optional() }) })`.
    - `healthResponseSchema`: `z.object({ status: z.literal('ok'), timestamp: z.number(), version: z.string() })`.
    - `searchQuerySchema`: `z.object({ q: z.string().min(1).max(500), limit: z.coerce.number().int().min(1).max(100).default(20), userId: z.string().optional() })`.
    - `searchResultSchema`: `z.object({ id: z.string(), content: z.string(), type: z.enum(['capture', 'commit', 'project']), projectId: z.string().nullable(), rank: z.number(), createdAt: z.string() })`.

    **`packages/shared/src/types/index.ts`:**
    - Derive TypeScript types from all schemas using `z.infer<>`.
    - Export: `Capture`, `CreateCapture`, `UpdateCapture`, `ListCapturesQuery`, `Project`, `ProjectListQuery`, `ApiError`, `HealthResponse`, `SearchQuery`, `SearchResult`.

    **`packages/shared/src/index.ts`:**
    - Barrel export everything from schemas and types.

    All schemas include `userId` as optional field per PLAT-02 (multi-user future-proofing).
