1. **Create packages/web/src/hooks/useIdeation.ts** — Custom hook managing full ideation lifecycle:

   - State: `idea` (string), `sessionId` (string|null), `status` ('idle'|'submitting'|'running'|'complete'|'error'), `stages` (Map of stage -> 'pending'|'running'|'complete'|'error'), `artifacts` (array of { id, stage, title, excerpt, artifactPath }), `activeStage` (string|null), `conversationEvents` (array of AgentSSEEvent from active stage), `error` (string|null)

   - `startIdeation(idea: string)` — POST to `/api/ideation/start` with { idea }. On success, set sessionId and status to 'running'. Open EventSource to `/api/ideation/stream/${sessionId}`.

   - EventSource message handler (follows useAgentStream.ts pattern):
     - `ideation:stage:start` -> update stages Map, set activeStage, clear conversationEvents
     - `ideation:stage:event` -> append to conversationEvents (forward inner agent events for rendering)
     - `ideation:stage:complete` -> update stages Map
     - `ideation:stage:artifact` -> fetch artifacts from `/api/ideation/artifacts/${sessionId}` and update artifacts array
     - `ideation:stage:error` -> update stages Map with 'error', set error message
     - `ideation:pipeline:complete` -> set status to 'complete', close EventSource

   - Cleanup: close EventSource on unmount or session change.

   - Return: `{ idea, setIdea, startIdeation, sessionId, status, stages, artifacts, activeStage, conversationEvents, error }`
