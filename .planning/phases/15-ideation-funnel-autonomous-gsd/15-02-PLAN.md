2. **Create packages/api/src/autonomous/gate-manager.ts** (per D-09, D-10, D-11):

   - `class GateManager` with:
     - `pendingGates: Map<string, { resolve: (response: string) => void; reject: (error: Error) => void; gate: DecisionGateRow }>`
     - `createGate(gate: { id: string; autonomousRunId: string; title: string; description: string; options: string; blocking: boolean }): Promise<string>` — Inserts into decisionGates table. Creates Promise. Returns the gate ID. Emits `autonomous:gate:created` on pipelineBus.
     - `resolveGate(gateId: string, response: string): boolean` — Finds pending gate, resolves its Promise with the response, updates DB (response, respondedAt). Returns true if found, false otherwise. Emits `autonomous:gate:resolved` on pipelineBus.
     - `getPendingGates(runId: string): DecisionGateRow[]` — Returns all unresolved gates for a run from DB.
     - `cleanup(runId: string): void` — Rejects all pending gates for a run (used on cancellation/error).
   - Export singleton: `export const gateManager = new GateManager()`

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser -> API | User-supplied projectPath, gateId, scaffold name cross into server |
| API -> Filesystem | Scaffold writes new directories, autonomous reads project state |
| API -> Child Process | GSD CLI commands spawned as child processes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-06 | Tampering | autonomous.ts POST /launch | mitigate | Validate projectPath starts with HOME, verify directory exists on filesystem before spawning |
| T-15-07 | Tampering | scaffold.ts POST /scaffold | mitigate | Validate name with regex /^[a-z0-9-]+$/ — no path traversal, no shell metacharacters. Reject if target path already exists. |
| T-15-08 | Denial of Service | executor.ts | mitigate | Limit to 1 concurrent autonomous run. maxBudgetUsd: 10.0 per phase, maxTurns: 200. |
| T-15-09 | Elevation of Privilege | executor.ts child_process | mitigate | Use spawn (not exec) to avoid shell injection. Arguments passed as array, not concatenated string. |
| T-15-10 | Tampering | gate-manager.ts resolveGate | accept | Single-user system, gate IDs are nanoid (unguessable). No auth needed. |
</threat_model>
