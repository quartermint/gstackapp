1. **Create packages/web/src/hooks/useAutonomous.ts** — Hook managing autonomous execution:

   - State: `runId` (string|null), `status` ('idle'|'launching'|'running'|'complete'|'failed'), `phases` (array of { number, name, status: 'pending'|'running'|'complete'|'failed'|'blocked' }), `commits` (array of { phase, hash, message, timestamp }), `agentSpawns` (array of { phase, agentId, role }), `completionSummary` ({ totalPhases, totalCommits, elapsedMs } | null), `error` (string|null)

   - `launchExecution(projectPath: string, ideationSessionId?: string)` — POST to `/api/autonomous/launch`. On success, set runId and status to 'running'. Open EventSource to `/api/autonomous/stream/${runId}`.

   - EventSource handler:
     - `autonomous:phases:discovered` -> set phases array with all 'pending'
     - `autonomous:phase:start` -> update phase status to 'running'
     - `autonomous:phase:complete` -> update phase status to 'complete'
     - `autonomous:phase:failed` -> update phase status to 'failed'
     - `autonomous:commit` -> append to commits array
     - `autonomous:agent:spawn` -> append to agentSpawns array
     - `autonomous:gate:created` -> delegated to useDecisionGates (shared via context or callback)
     - `autonomous:gate:resolved` -> delegated to useDecisionGates
     - `autonomous:complete` -> set completionSummary, status to 'complete', close EventSource
     - `autonomous:error` -> set error, status to 'failed'

   - `cancelExecution()` — POST to `/api/autonomous/${runId}/cancel`. Close EventSource.

   - Return: `{ runId, status, phases, commits, agentSpawns, completionSummary, error, launchExecution, cancelExecution }`

2. **Create packages/web/src/hooks/useDecisionGates.ts** — Hook managing decision gate queue:

   - State: `gates` (array of { id, title, description, options, blocking, response: string|null }), `pendingCount` (number)

   - `addGate(gate)` — Append to gates array. Blocking gates sort first (per D-11: blocking decisions surface prominently).

   - `respondToGate(gateId: string, response: string)` — POST to `/api/autonomous/${runId}/gate-response` with { gateId, response }. On success, update gate.response and remove from pending.

   - `resolveGate(gateId: string)` — Called when SSE confirms resolution. Updates gate state.

   - Return: `{ gates, pendingCount, addGate, respondToGate, resolveGate }`
