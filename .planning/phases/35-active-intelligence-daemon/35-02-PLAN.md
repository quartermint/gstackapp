<objective>
Build the "Previously on..." narrative generator and wire it end-to-end from LM Studio through API to dashboard.

Purpose: Per D-04, each project card shows an AI-generated context restoration narrative summarizing recent commits, captures, and session outcomes. Per D-01, this extends the local LLM from passive probe to active inference. Per D-05, narratives are cached with 1h TTL and never block API responses.

Output: Narrative generator service, intelligence API routes, React hook + panel component, hero card integration.

           summary: z.string().describe("2-3 sentence TV-recap style narrative of recent project activity"),
