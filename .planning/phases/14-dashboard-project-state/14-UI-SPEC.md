### Accent Reserved For (Phase 14)

- "Start Session" CTA button on project cards

### Phase 14 Status Colors

| Status | Color | Indicator |
|--------|-------|-----------|
| Active project | `#C6FF3B` (accent) | Solid dot, left card edge |
| Stale project | `#FFB020` (warning/flag) | Hollow ring dot, left card edge |
| Ideating project | `#B084FF` (design stage) | Solid dot, left card edge |
| Healthy service | `#2EDB87` (pass) | Filled circle in Mac Mini panel |
| Degraded service | `#FFB020` (flag) | Filled circle in Mac Mini panel |

### Staleness Threshold

- Active: any git activity or GSD state change within 3 days
- Stale: no activity for 3+ days AND has uncommitted changes
- Ideating: has design docs in `~/.gstack/projects/` but no `.planning/` directory

Source: DESIGN.md color palette, CONTEXT.md D-09, Claude's Discretion (stale threshold)

## Layout Contract

### Dashboard Grid (Main Content)

```
+--[Sidebar 220px]--+--[Main Content Area]--+
|                    |                       |
| gstackapp          | [Section: Projects]   |
| ---                | [Card] [Card] [Card]  |
| > Dashboard        | [Card] [Card] [Card]  |
|   Sessions (0)     |                       |
|   ---              | [Section: Infra]      |
|   Trends           | [Mac Mini] [Funnel]   |
|   Repositories     |                       |
|   Design Docs      | [Section: PR Reviews] |
|   ---              | [Existing v1.0 feed]  |
|   v2.0.0           |                       |
+--------------------+-----------------------+
```

### Project Cards Grid

- CSS Grid: `grid-template-columns: repeat(auto-fill, minmax(280px, 1fr))`
- Gap: 24px (lg token)
- Card padding: 16px (md token)
- Card border: 1px solid `#2A2F3A` (border)
- Card border-radius: 8px (radius-md)
- Card hover: background shifts to `#1A1D24` (surface-hover), border to `#3D4350` (border-focus), transition 150ms ease-out
- Card click: opens session tab for that project (D-11)

### Project Card Anatomy

```
+--[Status Dot]--[Project Name]--[Activity Badge]--+
|                                                    |
| Phase 3 of 5 — Pipeline Engine                    |
| main  +4 -2  ·  3 uncommitted                    |
|                                                    |
| Last activity: 2h ago                              |
+----------------------------------------------------+
```

- **Row 1**: Status dot (4px circle, left-aligned) + project name (Subheading 18px/500) + activity badge (right-aligned, Small 13px)
- **Row 2**: GSD phase summary (Body 15px/400, text-muted). Format: "Phase {n} of {total} -- {phase name}". If no `.planning/`, show "No GSD" in mono label style.
- **Row 3**: Git branch + diff summary + uncommitted count (Small 13px/400, text-muted). Format: `main  +4 -2  ·  3 uncommitted`. Uncommitted count uses accent color if > 0.
- **Row 4**: Last activity timestamp (Caption 12px/500, text-muted). Relative format via date-fns ("2h ago", "3 days ago").

### Project Sorting (Claude's Discretion)

Default sort order:
1. Active projects first, then stale, then ideating
2. Within each group: sort by last activity (most recent first)

User override: none for Phase 14 (manual sorting deferred)

### Sidebar Extension

Extend existing Sidebar.tsx with:
- **Sessions section** between Dashboard and Trends: lists open session tabs with project name, truncated to sidebar width. Clicking switches to that session view.
- **Design Docs** nav item below Repositories
- Session items show a close button (x) on hover, right-aligned

### Cmd+K Command Palette

- Overlay: centered, 480px wide, max-height 400px
- Background: `#13161C` (surface) with `#2A2F3A` border
- Backdrop: `rgba(11, 13, 17, 0.7)` blur(8px)
- Search input: full width, 15px body text, no visible border, placeholder "Search projects, sessions, docs..."
- Results list: 8px vertical padding between groups
- Result item: 40px height, 12px horizontal padding, Body 15px, hover `#1A1D24`
- Selected result: `rgba(198, 255, 59, 0.06)` background, accent text
- Groups: "Projects", "Sessions", "Design Docs" — group label in Caption 12px/500 text-muted uppercase
- Keyboard: Arrow keys navigate, Enter selects, Escape closes
- Animation: fade-in 150ms ease-out on open, fade-out 100ms ease-in on close

### Mac Mini Status Panel

- Card-style panel below project grid
- Title: "Infrastructure" (Heading 24px)
- Sub-items as horizontal row of status cards:
  - Each: service name (Body 15px), status dot (health color), endpoint URL (Mono label 11px)
  - Tailscale Funnel endpoints shown as clickable mono-label URLs

### Design Doc Browser

- Triggered via sidebar nav item "Design Docs"
- List view: project name + doc title + date, sorted by recency
- Each item: surface card style, click to expand/preview
- Preview: renders markdown content in body text, monospaced code blocks in JetBrains Mono 14px
- Empty state when no design docs exist

### Worklog Carryover View

- Integrated as a collapsible section on dashboard (below projects, above infra)
- Title: "Carryover Items" with count badge
- Each item: project name (Caption), carryover text (Body), staleness indicator
- Staleness: days since item was logged, color-coded (text-muted < 3 days, warning 3-7 days, destructive > 7 days)

### PR Review Integration (PREV-01)

- Existing v1.0 dashboard (PipelineHero + PRFeed + PRDetail) accessible as "PR Reviews" nav item in sidebar
- When selected, replaces main content area with existing v1.0 view
- No visual changes to existing components

## Interaction States

### Project Card

| State | Visual |
|-------|--------|
| Default | Surface background, border, static content |
| Hover | Surface-hover background, border-focus, cursor pointer, 150ms transition |
| Loading | Skeleton: 280px x 140px rounded-md with pulse animation |
| Error (single card) | Card renders with text-muted "Could not read project state" in body area |

### Command Palette

| State | Visual |
|-------|--------|
| Closed | Not rendered |
| Open (empty query) | Shows recent projects (up to 5) |
| Open (with query) | Filtered results grouped by type |
| No results | "No results for '{query}'" in text-muted, centered |

### Sidebar Session Tab

| State | Visual |
|-------|--------|
| Default | Text-muted, no background |
| Hover | Text-primary, surface-hover background, close (x) button appears |
| Active | Accent text, accent-muted background |

### Mac Mini Service

| State | Visual |
|-------|--------|
| Healthy | Green dot + service name + endpoint URL |
| Degraded | Warning dot + service name + "degraded" label |
| Down | Red dot + service name + "unreachable" label |
| Loading | Skeleton dots + skeleton text lines |

## Motion Contract

| Animation | Duration | Easing | Trigger |
|-----------|----------|--------|---------|
| Card hover | 150ms | ease-out | Mouse enter/leave |
| Command palette open | 150ms | ease-out | Cmd+K keydown |
| Command palette close | 100ms | ease-in | Escape or click outside |
| Card grid load | 400ms stagger (50ms per card) | ease-out | Initial data load |
| Status dot pulse (stale) | 2s infinite | ease-in-out | Stale project indicator |
| Carryover section expand | 250ms | ease-in-out | Click toggle |

Anti-patterns (per DESIGN.md): No scroll-driven animation, no bounce, no parallax, no decorative motion.
