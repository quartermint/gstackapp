Visual spec (from UI spec):
- Overlay: fixed inset-0, backdrop `rgba(11, 13, 17, 0.7)` with `backdrop-blur-sm`
- Panel: centered, 480px max-width, max-h-[400px], bg-surface, border border-border, rounded-lg, overflow-hidden
- Search input: full width, p-4, text-[15px] font-body, bg-transparent, border-none, text-text-primary, placeholder "Search projects, sessions, docs..." in text-muted. Focus outline none.
- Results area: overflow-y-auto, max remaining height
- Group headers: px-3 py-2, font-body text-[12px] font-medium text-text-muted uppercase tracking-[0.06em]
- Result items: h-[40px] px-3 flex items-center, text-[15px] font-body, hover:bg-surface-hover, cursor-pointer
- Selected item: bg-accent-dim (`rgba(198, 255, 59, 0.06)`), text-accent

Data sources:
- `useProjects()` for project list -> group "Projects"
- `useSessions()` from parent props or hook -> group "Sessions"
- Design docs from `useDesignDocs()` -> group "Design Docs"

Search behavior:
- Filter all items by query (case-insensitive substring match on name/title)
- When query is empty, show recent projects (up to 5)
- Group results: "Projects" first, then "Sessions", then "Design Docs"
- Hide empty groups

Keyboard navigation:
- `ArrowDown` / `ArrowUp` move selectedIndex through flattened results list
- `Enter` triggers action for selected item (navigate to project view, switch to session, navigate to design docs)
- `Escape` closes palette
- Click outside (on backdrop) closes palette
- Auto-focus search input on open

Actions:
- Project selected: call `onSelectProject(name)` then `onClose()`
- Session selected: call `onSelectSession(id)` then `onClose()`
- Design doc selected: call `onNavigate('design-docs')` then `onClose()`
