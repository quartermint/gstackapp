```typescript
function inferProviderFromModel(model: string): string | undefined {
  if (model.startsWith('claude-')) return 'anthropic'
  if (model.startsWith('gemini-')) return 'gemini'
  // Codex-specific models route to codex provider (sandbox-capable)
  if (model.includes('codex')) return 'codex'

  if (model.startsWith('qwen')) return 'local'
  if (model.startsWith('gemma')) return 'local'  // Gemma 4 26B-A4B
  return undefined
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Harness -> OpenAI API | API key transmitted to OpenAI for GPT-5.x calls |
| Harness -> Codex CLI | API key injected into subprocess environment |
| Harness -> Mac Mini MLX | Internal Tailscale network, no auth needed |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-01 | Information Disclosure | CodexProvider subprocess | mitigate | Pass API key via SDK env injection (not CLI args). SDK handles this internally via `env: { CODEX_API_KEY }` constructor option. Verify key is not visible in `process.argv`. |
| T-13-02 | Elevation of Privilege | Codex sandbox | mitigate | Default to `workspace-write` sandbox mode. Never use `danger-full-access`. Scope workingDirectory to project root. Implemented in CodexProvider.runSandbox() options. |
| T-13-03 | Denial of Service | Codex subprocess | mitigate | Set explicit timeout (default 120s) on thread.run() via AbortController. Kill subprocess on timeout. Prevents runaway sandbox blocking harness. |
| T-13-04 | Information Disclosure | API keys in config | accept | Keys loaded from .env per existing pattern. .env is gitignored. Single-user system, no multi-tenant risk. |
</threat_model>
