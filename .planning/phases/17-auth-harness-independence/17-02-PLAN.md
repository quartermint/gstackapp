    **Operator routes** (packages/api/src/routes/operator.ts):

    `POST /request`: Zod schema: `{ whatNeeded: z.string().min(1).max(5000), whatGood: z.string().min(1).max(5000), deadline: z.string().optional() }`. Get user from `getUserScope(c)`. Insert into `operatorRequests` with nanoid id and user's userId. Insert audit trail entry with action 'request_submitted'. Return `201 { id, status: 'pending' }`.

    `GET /history`: Get user from `getUserScope(c)`. If role === 'operator', query `operatorRequests WHERE userId = user.userId ORDER BY createdAt DESC LIMIT 50`. If role === 'admin', query all requests `ORDER BY createdAt DESC LIMIT 100`. Return array of requests.

    `GET /request/:id`: Get user from `getUserScope(c)`. Load request by id. If operator and request.userId !== user.userId, return 403. Return full request with audit trail entries.

| GET /api/operator/history -> Browser | Must only return current user's data (operator) or all (admin) |
