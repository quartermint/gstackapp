**Step 2: Create `packages/api/src/pipeline/sandbox.ts`.**

Implements CVE-2025-53109 prevention per D-17. The critical security pattern:
1. `resolve()` the candidate path from sandboxRoot + requestedPath
2. `realpathSync()` FIRST to resolve any symlinks to their true target
3. THEN check that the resolved real path starts with the real sandbox root
4. If realpathSync throws (file not found), throw "File not found"
5. If prefix check fails, throw "Access denied: path escapes sandbox"

```typescript
// packages/api/src/pipeline/sandbox.ts
import { realpathSync } from 'node:fs'
import { resolve } from 'node:path'

export function validatePath(requestedPath: string, sandboxRoot: string): string {
  const candidatePath = resolve(sandboxRoot, requestedPath)
  let realPath: string
  try {
    realPath = realpathSync(candidatePath)
  } catch {
    throw new Error(`File not found: ${requestedPath}`)
  }
  const realRoot = realpathSync(sandboxRoot)
  if (!realPath.startsWith(realRoot + '/') && realPath !== realRoot) {
    throw new Error('Access denied: path escapes sandbox')
  }
  return realPath
}
```

**Step 1: Create `packages/api/src/pipeline/clone.ts`.**

Per D-15, D-16. Exports:
- `cloneRepo(installationId: number, repoFullName: string, headRef: string): Promise<string>` -- Returns path to clone directory.
  1. Create temp dir: `mkdtemp(join(tmpdir(), 'gstack-' + repoFullName.replace('/', '-') + '-'))`
  2. Get installation Octokit: `getInstallationOctokit(installationId)`
  3. Get installation access token for the clone URL. Use: `const { token } = await (octokit as any).auth({ type: 'installation', installationId })`. If this fails, fall back to calling `POST /app/installations/{installationId}/access_tokens` via the Octokit REST client.
  4. Build clone URL: `https://x-access-token:${token}@github.com/${repoFullName}.git`
  5. Shallow clone: `simpleGit().clone(cloneUrl, clonePath, ['--depth', '1', '--single-branch', '--branch', headRef])`
  6. Remove symlinks: `execFileSync('find', ['.', '-type', 'l', '-delete'], { cwd: clonePath, timeout: 5000 })` -- catch errors (no symlinks = common case)
  7. Return clonePath

- `cleanupClone(clonePath: string): Promise<void>` -- `rm(clonePath, { recursive: true, force: true })`, log errors but don't throw.

Import `simpleGit` from `simple-git`, `execFileSync` from `node:child_process`, `rm, mkdtemp` from `node:fs/promises`, `join` from `node:path`, `tmpdir` from `node:os`, `getInstallationOctokit` from `../github/auth`.

**Step 2: Create `packages/api/src/pipeline/filter.ts`.**

Per D-08, D-09, D-10. Exports:
- `shouldRunStage(stage: Stage, files: PrFile[]): boolean`

Import `Stage` type from `@gstackapp/shared`.

Define `PrFile` interface: `{ filename: string; status: string; additions: number; deletions: number }`. Export the PrFile type.

Logic:
- `eng`, `qa`, `security`: always return `true`
- `ceo` returns true if ANY of:
  - `files.some(f => f.status === 'added')` (new files)
  - `files.some(f => f.filename.includes('architect') || f.filename.includes('config') || /^(docker|\.github|\.ci)/i.test(f.filename))` (architecture changes)
  - `files.some(f => /package\.json|requirements\.txt|go\.mod|Cargo\.toml|Gemfile/i.test(f.filename))` (dependency changes)
  - `files.length > 10 || files.reduce((sum, f) => sum + f.additions + f.deletions, 0) > 500` (large PR)
- `design` returns true if `files.some(f => /\.(css|scss|less|styled|tsx|jsx)$/i.test(f.filename) || /component|style|theme|layout|ui|design/i.test(f.filename))`
- Default: return `true`
