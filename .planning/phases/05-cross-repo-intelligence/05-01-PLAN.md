Build the embedding infrastructure and cross-repo search layer for finding similarity matching.

Output: Embedding module (5 files), orchestrator wiring, sqlite-vec loaded at startup, comprehensive test coverage.

        embedding float[1024] distance_metric=cosine,
        repo_full_name TEXT,
        stage TEXT,
        severity TEXT,
        +title TEXT,
        +description TEXT,
        +file_path TEXT
      )

    - `insertFindingEmbedding(db, findingId, embedding, metadata)` -- single insert with `embedding.buffer` (Pitfall 5: pass .buffer not Float32Array)

    - Post-query filter: exclude findings where feedbackVote = 'down' by joining back to findings table
    - Export `CrossRepoMatch` interface: { finding_id, title, description, file_path, repo_full_name, distance, stage, severity }
