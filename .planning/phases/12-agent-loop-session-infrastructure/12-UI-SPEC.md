## Layout Contract

### Session View Layout (within Shell)

```
+--[Sidebar 220px]--+--[Conversation]--+--[Artifact Panel 480px]--+
|                    |                  |                           |
| gstackapp          |  MessageList     |  Artifact content         |
| ──────────         |  (scrollable)    |  (code, docs, GSD state)  |
| Dashboard          |                  |                           |
| Trends             |                  |                           |
| Repos              |                  |                           |
| ──────────         |                  |                           |
| Sessions           |  ──────────────  |                           |
|  > Session 1 (act) |  InputArea       |                           |
|    Session 2       |  [___________]   |                           |
|    Session 3       |  [Send]          |                           |
| + New Session      |                  |                           |
+--------------------+------------------+---------------------------+
```

- Sidebar: 220px fixed (existing). Session list added as new section below nav separator.
- Conversation area: flex-1, fills remaining width. When artifact panel is closed, conversation takes full remaining width.
- Artifact panel: 480px fixed width, right side. Toggled open/closed. Closed by default. Opens when AI produces an artifact or user clicks one.
- Input area: pinned to bottom of conversation column. Min height 48px (single line + padding). Max height 200px (auto-resize up to ~8 lines).
- Bottom strip: 40px, spans conversation + artifact panel (existing behavior).

### Message Layout

- Messages are left-aligned, full-width within conversation column. Not centered bubbles.
- User messages: no background, `text-primary` color, 15px Geist. Left-aligned flush with conversation padding.
- AI messages: `bg-surface` background, `rounded-lg` (12px), padding 16px. Left-aligned flush with conversation padding.
- Message groups (consecutive same-author): 8px gap between messages in same group, 24px gap between groups.
- Timestamps: shown on hover per message, or once per message group. 12px caption, `text-muted`.
- Maximum message content width: 720px (readable line length within conversation column).

### Tool Call Display

- Inline within AI message flow, between text blocks.
- Collapsed by default: shows tool name (mono label, 11px uppercase) + status badge (caption, 12px) + duration.
- Expanded on click: shows arguments (code block, 14px JetBrains Mono) and result (code block or formatted output).
- Status badge colors: RUNNING (`#36C9FF` bg at 12% opacity, `#36C9FF` text), success (`#2EDB87`), error (`#FF5A67`), skipped (`#6F7C90`).
- Border-left accent: 2px solid, color matches status.

## Interaction Contract

### Message Sending
- Enter sends message (single-line mode). Shift+Enter adds newline.
- Send button: accent background (`#C6FF3B`), dark text (`#0B0D11`), rounded-md (8px). Disabled state: 20% opacity.
- Input clears on send. Focus returns to input after AI response completes.

### Streaming
- AI response streams token-by-token via SSE (reuse existing `useSSE` hook pattern).
- Streaming cursor: vertical bar, accent color, 1s blink animation (opacity 0 to 1, ease-in-out, infinite).
- Message bubble appears immediately on first token. Auto-scroll follows streaming output.
- User can scroll up during streaming — auto-scroll disengages. "Scroll to bottom" pill appears at bottom of message list.

### Tool Execution (inline)
- Tool calls appear inline in the message stream as they execute.
- Running state: pulsing border-left accent (reuse `pulse-glow` keyframe pattern, adapted for border).
- Completed: static border-left, collapsed view with result summary.
- Multiple consecutive tool calls stack vertically with 4px gap.

### Session Management
- Session list in sidebar: most recent first. Each item shows title (truncated to sidebar width) + relative timestamp ("2h ago").
- Active session: accent text + accent-muted background (same pattern as active nav item).
- "New Session" button: bottom of session list, accent text, `+` prefix. Creates session and navigates to it.
- Session title: auto-generated from first user message (first 50 chars). Editable on double-click (inline edit, Enter to confirm, Escape to cancel).

### Session Resume
- On session select: show SkeletonText placeholders while loading message history from SQLite.
- Messages load in reverse-chronological batches (most recent 50 first), with infinite scroll up for history.
- Compression indicator: thin bar (2px height) at top of message list, warning color (`#FFB020`), appears when compression pipeline has been active. Tooltip: "Context compressed to maintain coherence."

### Artifact Panel
- Opens with slide-in from right, 250ms ease-out.
- Close button: top-right corner, `X` in text-muted, 32px hit target.
- Content renders based on artifact type: code (syntax highlighted with JetBrains Mono), markdown (rendered), GSD state (structured display).
- Panel has its own vertical scroll, independent of conversation scroll.

### Empty States
- No sessions: EmptyState component centered in conversation area.
- No messages in new session: input area visible, conversation area shows subtle prompt text.

## State Catalog

| State | Visual Treatment |
|-------|-----------------|
| Idle (input focused, no activity) | Input area border: `border-border`. Send button: disabled (20% opacity). |
| Composing (user typing) | Input auto-resizes. Send button: enabled (full accent). Character in input. |
| Sending | Input locked (disabled). Send button shows sending state (subtle pulse). |
| Streaming (AI responding) | Streaming cursor visible. Auto-scroll active. Input disabled until complete. |
| Tool executing | ToolCallBlock with RUNNING status, pulsing border-left. |
| Tool complete | ToolCallBlock collapsed, static border-left, status badge. |
| Session loading | SkeletonText placeholders in message list. |
| Session loaded | Messages rendered, input focused. |
| Session empty | Placeholder text in input area. Conversation area empty. |
| No sessions | EmptyState component centered. |
| Compression active | 2px warning-color bar at top of message list. |
| Artifact open | Artifact panel slides in from right. Conversation area narrows. |
| Scrolled up (during stream) | "New messages" pill appears at bottom. |
| Connection lost | Error banner at top of conversation: "Connection lost. Reconnecting..." (error semantic colors). |
