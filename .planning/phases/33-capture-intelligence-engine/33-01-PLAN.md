<objective>
Build the core capture intelligence pipeline: few-shot categorization with user-correctable examples, multi-pass extraction, LM Studio fallback, and correction tracking.

Purpose: Replace the zero-shot Gemini categorizer with a few-shot system that learns from user corrections and extracts structured items (action items, ideas, questions) from every capture. This is the foundation all other Phase 33 plans build on.

Output: New DB tables (few_shot_examples, correction_stats), new captures columns (source_type, source_metadata, extractions, grounding_data), few-shot-categorizer.ts service, updated enrichment pipeline, correction flow in PATCH route.
</objective>
