       - Import `DiffMatchPatch` from "diff-match-patch-es"
       - Export type `GroundingTier = "exact" | "fuzzy" | "ungrounded"`
       - Export interface `GroundedSpan { text: string; startOffset: number; endOffset: number; tier: GroundingTier; extractionIndex: number }`
       - Export `groundExtraction(sourceText: string, extractedText: string, extractionIndex: number, fuzzyThreshold = 0.6): GroundedSpan | null`
         - Try exact match first: `sourceText.toLowerCase().indexOf(extractedText.toLowerCase())`
         - If found: return span with tier "exact", using original case from sourceText
         - If not found: use `new DiffMatchPatch()`, set `dmp.matchThreshold = fuzzyThreshold`, call `dmp.matchMain(sourceText.toLowerCase(), extractedText.toLowerCase(), 0)`
         - If fuzzy match found (index >= 0): return span with tier "fuzzy"
         - If no match: return null
       - Export `groundAllExtractions(sourceText: string, extractions: Array<{ type: string; text: string; confidence: number }>): GroundedSpan[]`
         - Map over extractions, call groundExtraction for each, filter out nulls
         - Return array of GroundedSpan (serializable to JSON for storage in grounding_data column)

         - Fetch project list via listProjects(db)

         - Compare result.projectSlug with example.correctProjectSlug
         - If mismatch: `console.warn(\`[prompt-validator] Example mismatch: "${example.captureContent.slice(0, 50)}..." expected ${example.correctProjectSlug}, got ${result.projectSlug}\`)`
