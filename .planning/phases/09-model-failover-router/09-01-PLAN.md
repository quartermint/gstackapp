    - getHarnessDb(dbPath) opens SQLite connection with WAL mode, returns Database instance
    - getHarnessDb(undefined) returns null (no DB configured)
    - getHarnessDb(dbPath) creates data directory if it does not exist
    - tokenUsage Drizzle schema has columns: id (auto-increment), provider (text), timestamp (integer), inputTokens (integer), outputTokens (integer), costEstimate (real, nullable), stage (text, nullable)
    - tokenUsage schema has index on (provider, timestamp)
    - UsageBuffer.record() adds record to internal buffer (does not touch DB)
    - UsageBuffer.flush() writes all buffered records to DB in a transaction
    - UsageBuffer.flush() clears the buffer after successful write
    - UsageBuffer.flush() on DB error: logs warning, does NOT throw, buffer is NOT lost (records stay for next flush attempt)
    - UsageBuffer.flush() with empty buffer is a no-op
    - UsageBuffer.start() begins periodic flush interval
    - UsageBuffer.shutdown() clears interval and does a final flush
    - UsageBuffer works without DB (null db): record() buffers silently, flush() is a no-op
